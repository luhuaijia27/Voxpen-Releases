## 安装说明

### Windows
1. 下载并解压 `Voxpen_Setup_v1.0.0.zip`
2. 运行 `Voxpen_Setup.exe`，按提示完成安装

### macOS（仅支持 Apple Silicon M1/M2/M3/M4）
1. 下载 `Voxpen_v1.0.0_macOS_arm64.dmg`
2. 双击挂载，运行其中的安装脚本
3. 首次启动需在系统设置中授权：辅助功能、输入监控、麦克风
